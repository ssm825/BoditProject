## 과제 요건

#### 공통

- 센서 목록
- 그래프
- csv 파일 export
- 반응형 레이아웃 구현

<br>

#### 센서리스트(첫번째 화면)

- 센서 리스트 출력
- 내림차순, 올림차순 정렬
- 다중 필터링
- hover 기능 구현
- 20%이하 배터리 부족 표시
  
<br>

- 입력 데이터는 API요청
